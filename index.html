隐私政策

适用范围：开发者发布的全部桌面应用程序。

最后更新日期：2026年4月19日

生效日期：2026年4月19日

一、信息收集与使用

1.1 我们不收集任何个人信息

我们开发的应用程序为纯本地工具，不会收集、存储、传输任何能够识别个人身份的信息，包括但不限于：

  • 姓名、身份证号、电话号码、电子邮件地址
  • 位置信息、IP地址、浏览器指纹
  • 设备标识符、MAC地址、IMEI
  • 使用习惯、操作记录、点击行为

1.2 应用程序访问范围

应用程序可能访问以下本地资源，但这些操作完全在用户设备本地执行，数据不会离开用户设备：

访问类型目的数据去向
文件系统访问打开和保存用户文件仅本地，不传输
剪贴板访问支持复制粘贴功能仅本地，不传输
系统通知向用户展示操作结果提示仅本地，不传输

二、数据存储

2.1 本地存储

应用程序可能在用户设备本地存储以下数据：

  • 用户配置：如窗口大小、主题偏好、语言设置等。这些数据存储在应用程序的本地配置目录中，用于保存用户的个性化设置。
  • 临时文件：应用程序运行过程中产生的临时缓存文件，关闭应用后会被清理或由用户手动删除。
  • 最近打开文件记录（如有）：仅记录文件路径列表，方便用户快速访问，不包含文件内容。

2.2 数据删除

用户可以通过以下方式删除应用程序存储的本地数据：

  • 卸载应用程序（部分应用会在卸载时自动清理配置文件）
  • 在应用程序设置中手动清除数据或重置设置
  • 手动删除应用程序的配置目录（通常位于用户主目录下的隐藏文件夹中）

三、网络访问

3.1 完全离线运行

应用程序不需要也不使用网络连接即可运行全部功能。所有核心功能均为离线操作，数据不会发送到任何服务器。

3.2 自动网络请求

本应用程序不发起任何自动网络请求，包括：

  • 不进行任何数据上传或下载
  • 不使用分析、统计或埋点服务
  • 不连接任何第三方服务或API
  • 不显示任何广告或推送通知

四、第三方服务与SDK

无第三方SDK集成

本应用程序不集成任何第三方数据分析、广告、推送、登录或追踪SDK。

五、安全措施

虽然应用程序不收集任何个人数据，我们仍然采取了以下安全措施来保护用户：

  • 应用程序的代码发布前经过基本安全检查
  • 应用程序签名验证（Windows平台使用代码签名证书签名）
  • 仅请求应用程序运行所需的最小系统权限

六、政策更新

如果我们将来需要更新本隐私政策，将在 GitHub 仓库（json-forge-privacy）上发布最新版本，并通过以下方式通知用户：

  • 更新页面顶部的"最后更新日期"
  • 在应用程序的更新说明中标注隐私政策变更

七、联系我们

如果您对本隐私政策有任何疑问、意见或建议，请通过以下方式联系我们：

  • Microsoft Store 平台：通过 Microsoft Store 应用详情页的"联系支持"功能发送消息
  • GitHub：在我们的 GitHub 仓库提交 Issue 或 Pull Request

我们将在收到您的反馈后尽快回复，通常不超过30个工作日。
